## Design Principles

### Core Values

- **Intent First**: Understand purpose before implementation
- **Craft Foundations**: Master fundamentals before advanced techniques
- **Subtle Layering**: Build depth through careful composition
- **Infinite Expression**: Explore creative possibilities within constraints

### Key Guidelines

1. **Accessibility** - All interactive elements need proper labels, keyboard support, and focus states
2. **Forms** - Proper validation, labels, and error handling
3. **Animation** - Respect prefers-reduced-motion, use compositor-friendly properties
4. **Typography** - Use proper punctuation, text-wrap utilities
5. **Performance** - Optimize images, virtualize large lists
